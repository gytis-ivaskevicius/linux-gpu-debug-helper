[Polybar](https://github.com/polybar/polybar/wiki) is a fast and easy-to-use status bar.

## Getting started {#getting_started}

You can copy the example configation from
`/nix/store/``<hash>`{=html}`-polybar-``<version>`{=html}`/share/doc/polybar``<ins>`{=html}`/examples``</ins>`{=html}`/config.ini`
to `~/.config/polybar/config.ini` and launch with `polybar -c ~/.config/polybar/config.ini example`

This default configuration requires the icon font \'siji\' which can be installed as documented on \[Fonts\]
