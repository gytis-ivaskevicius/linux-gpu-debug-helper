This page strives to list all games which have a package available in the [official
repositories](official_repositories "official repositories"){.wikilink} or the [AUR](AUR "AUR"){.wikilink}. There are
many more Linux games available, which are not packaged. See [Gaming#Getting
games](Gaming#Getting_games "Gaming#Getting games"){.wikilink} for ways to obtain them.

For more about running games, related system configuration tips, see [Gaming](Gaming "Gaming"){.wikilink}. For an up to
date selection of games available in the AUR, try checking the [AUR \'game\'
keyword](https://aur.archlinux.org/packages/?SeB=k&K=game).
