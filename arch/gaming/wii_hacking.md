A few starting packages for hacking the Wii on Arch Linux. Also see [Wiibrew.org](https://wiibrew.org/) for a guide to
getting homebrew setup on the Wii.

## Native Coding {#native_coding}

- ```{=mediawiki}
  {{AUR|wiiload}}
  ```
  lets you upload games to the Homebrew Channel

- ```{=mediawiki}
  {{Pkg|wit}}
  ```
  \- tools to manipulate Wii and GameCube ISO images

## Wiimote

- ```{=mediawiki}
  {{AUR|cwiid}}
  ```
  \- a library in C for working with the Wii Remote.

## Emulation

- ```{=mediawiki}
  {{Pkg|dolphin-emu}}
  ```
  \- the dominant emulator for the Wii (also supports Gamecube).

- ```{=mediawiki}
  {{AUR|dolphin-emu-git}}
  ```
  \- development version
